    注解学习

    通俗的来讲：
注解概念：官方的解释解释
注释：自己的解释说明

    JDK1.5 添加

    注解的作用： 解释、说明

    间接作用：替代配置文件

    注解种类:

    • jdk官方注解(@override/@Deprecated/@SuppressWarnnings)
    • 自定义注解
    • 元注解(注解的注解)

    自定义注解:

    简单demo：
    /**
    *注解里只能定义8种基本数据类型，String，字节码，注解类型，枚举，和他们的数组
    */
    public @interface DemoAnno(){
    String value();
    int name();
    …
    }
    如果属性名为value，在使用的时候可以省略属性名@DemoAnno(“test”)=
    =@DemoAnno(value=”test”)

    元注解

    @Target ： 指定注解的可以加在那些位置

    @Retention ：注解作用的时机(注解默认出现在编译时期)
