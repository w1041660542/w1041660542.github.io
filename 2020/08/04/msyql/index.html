    安装方式 (3种)：

        通用二进制版:解压即用     
        rpm,yum版本:下载rpm包或者配置yum源  
        源码包 :编译安装,非常慢    
    ### 二进制版安装步骤

    - 1)清除历史环境

        

    rpm -qa  | grep  mariadb 
     
    yum remove mariadb-1ibg-y  

    - 2）创建用户和组

        useradd mysql -s /sbin/nologin  
        id mysql

    - 3)创建相关目录

        创建软件目录    
        #mkdir -p /app/database/    
        创建建数据目录    
        #mkdir -p /data/3306/    
        创建日志目录  
        #mkdir -p /binlog/3306/  

    4)设置权限

    #chown -R mysql.mysql /app/ /data/ /binlog 

    5）上传软件 到/app/database/  并解压

    用的secureCTR远程连接   
    #yum -y install lrzsz   
    rz 上传   
    sz 下载
    #tar xf mysql-5.7.28-linux-glibc2.12-x86_64.tar.gz
    建立软连接
    #ln -s mysql-5.7.28-linux-glibc2.12-x86_64 mysql

    6） 设置环境变量

    vim /etc/profile
    添加一行:
    #export PATH=/app/database/mysql/bin:$PATH
    生效配置
    source /etc/profile
    如果环境变量配错 无法使用vim
    执行 $export PATH=/usr/bin:/usr/sbin:/bin:/sbin:/usr/X11R6/bin 暂时可用

    7）初始化系统库表

    mysqld --initialize-insecure --user-mysql --basedir=/app/database/mysql --datadir= /data/3306/
    报错
    mysqld: error while loading shared libraries: libao. so.1：
    cannot open shared object file: No such file or directory
    解决
    yum install -y libaio-devel
    再次运行
    mysqld --initialize-insecure --user-mysql --basedir=/app/database/mysql --datadir= /data/3306/
    报错，2020-08-29T05:30:52.575467Z 0 [Warning] TIMESTAMP with implicit DEFAULT value is deprecated. Please use –explicit_defaults_for_timestamp server option (see documentation for more details).
    2020-08-29T05:30:52.578017Z 0 [ERROR] –initialize specified but the data directory has files in it. Aborting.
    2020-08-29T05:30:52.578052Z 0 [ERROR] Aborting
    执行 $ rm -rf /data/3306/*

    8)配置文件的设置

    cat > /etc/my.cnf <<EOF [mysqld] user=mysql basedir=/app/database/mysql datadir=/data/3306 server_id=6 port=3306 socket=/tmp/mysql.sock [mysql] socket=/tmp/mysql.sock EOF

    9)准备 MYSQL启动脚本

    $cd /app/database/mysql/support-files/
    拷贝mysq1的启动脚本至系统软件管理目录中
    $ cp mysql.server /etc/init.d/mysqld

    最后 启动mysql
    service mysqld start
    成功
